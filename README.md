# SistemaEscolar-System

## Descrição

Projeto desenvolvido como atividade acadêmica com o objetivo de aplicar conceitos de Programação Orientada a Objetos (POO) em Java. O sistema simula uma estrutura básica de gerenciamento escolar, incluindo entidades como alunos e turmas.

## Estrutura do Projeto

O projeto é composto pelos seguintes arquivos principais:

* **Aluno.java**: Representa a entidade aluno, contendo atributos e comportamentos relacionados.
* **Turma.java**: Responsável pela modelagem de turmas e suas operações.
* **Main.java**: Classe principal que executa o sistema e demonstra seu funcionamento.

## Objetivo

Aplicar na prática conceitos fundamentais de POO, como:

* Encapsulamento
* Abstração
* Organização em classes
* Relacionamento entre objetos
